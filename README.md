# Shop Hub ZA (Netlify + Next.js)

## Deploy (recommended)
1. Unzip this project
2. Upload to GitHub (new repo)
3. Netlify -> Add new site -> Import from Git (GitHub)
4. Build command: npm run build
5. Node version: set env var NODE_VERSION=20 (Netlify Site settings -> Environment variables)
6. Deploy

## Quick MVP content
Edit `src/data/featured_deals.json` to add real products and your affiliate links while waiting for API approvals.
